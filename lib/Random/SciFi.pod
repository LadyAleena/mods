=head1 NAME

B<Random::SciFi> returns a random I<Hitckhikers Guide to the Galaxy> (HHGTTG) sector, a random I<Men in Black> (MiB) agent id, or a random I<Star Trek> Borg designation.

=head1 VERSION

This document describes Random::SciFi version 1.000.

=head1 SYNOPSIS

  use Random::SciFi qw(random_HHGTTG_sector random_MIB_agent);

  my $random_HHGTTG_sector = random_HHGTTG_sector(); # returns a random sector based on HHGTTG sectors mentioned.
  my $random_MIB_agent     = random_MIB_agent();     # returns a random MiB agent, either single or double letter.

=head1 DESCRIPTION

Random::SciFi returns a random I<Hitckhikers Guide to the Galaxy> (HHGTTG) sector or a random I<Men in Black> (MiB) agent id.

=head2 random_HHGTTG_sector

  random_HHGTTG_sector();

C<random_HHGTTG_sector> returns a random sector of the universe in I<The Hitchhikers Guide to the Galaxy> format.

The returned string starts with two random uppercase letters of the English alphabet, then is given a (for lack of a better term) type, one more uppercase letter of the English alphabet, and ending with a spelled out letter of the Greek alphabet. In the HHGTTG, Earth's sector is C<ZZ Plural Z Alpha>. Here are some sample sectors generated by this function.

  QJ Passive X Chi
  IO Heavy Z Nu
  MV Single P Mu
  PB Over Z Omicron
  IR Plural O Iota

=head2 random_MIB_agent

  random_MIB_agent();

C<random_MIB_agent> returns an identifier in the style of the I<Men in Black> frachise.

The returned string is one or two random uppercase letters of the English alphabet. Agent identifiers in the film series include: K, J, Z, O, L, M, H, C, and AA. Z is pronounced Zed.

=head2 random_Borg

  random_Borg();

C<random_Borg> returns a Borg designation based on the Borg designations used in the I<Star Trek> franchise.

The returned string is the number of the Borg in the group, the amount of Borg in the group, and the group's Unimatrix. The maximum group size has been set to twelve and the maximum Unimatrix is 100. Here are some sample Borg generated by this function.

  Eight of Eleven of Unimatrix 56
  Seven of Seven of Unimatrix 32
  One of Twelve of Unimatrix 97
  Eight of Twelve of Unimatrix 92
  Two of Twelve of Unimatrix 51

=head1 DEPENDENCIES

Random::SciFi depends on L<Random::Alpha>, L<Games::Dice>, L<Lingua::EN::Inflect>, and L<Exporter>.

Random::Alpha depends on L<Fancy::Rand> and L<Util::Data>.
